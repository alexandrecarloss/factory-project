## 🚀 Start in 3 Steps

### Step 1: Docker Compose Up

```bash
cd factory-project
docker-compose up --build
```

Or use quick start script:

```bash
./start.sh          # Linux/Mac
.\start.bat         # Windows
```

### Step 2: Open Browser

```
Frontend:  http://localhost:3000
Backend:   http://localhost:8080
Database:  localhost:5432
```

### Step 3: Test Features

1. Add Raw Materials
2. Add Products with Composition
3. Click "Optimize Production"
4. See Suggestions!
